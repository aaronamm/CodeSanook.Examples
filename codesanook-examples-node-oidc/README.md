### required packages
- yarn init -y
- yarn add express ejs

### Required setup files
- .babelrc
- .editorconfig
- .eslintrc.yml
- webpack.config.js
- tsconfig.json

### To run a project
- yarn run build
- yarn run serve
- Open a browser and navigate to http://localhost:3000
